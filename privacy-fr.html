Politique de confidentialité

Cette politique de confidentialité explique comment BOMEDYS NV traite les données à caractère personnel lorsque vous utilisez ce store locator Joie.

Ce store locator est conçu de manière à limiter autant que possible le traitement de données personnelles. Vous ne devez pas créer de compte et BOMEDYS NV ne crée pas de profils marketing individuels via ce site web.

Quelles données peuvent être traitées ?

Selon la manière dont vous utilisez le site web, les données suivantes peuvent être traitées :

  • la localisation que vous partagez volontairement via votre navigateur lorsque vous cliquez sur « Utiliser ma position actuelle » ;
  • un code postal, une ville, une commune ou une adresse introduite dans la fonction de recherche ;
  • des données techniques nécessaires à l’affichage du site web, telles que l’adresse IP, le type de navigateur, le type d’appareil, la date et l’heure de la visite ;
  • votre préférence linguistique, qui peut être mémorisée localement dans votre navigateur ;
  • les données que vous fournissez vous-même lorsque vous contactez BOMEDYS NV, par exemple par e-mail.

Utilisation de votre localisation

Lorsque vous utilisez votre position actuelle, votre navigateur vous demande d’abord votre autorisation. Sans votre autorisation, ce site web ne peut pas utiliser votre localisation précise.

La localisation partagée est utilisée pour trier les points de vente par distance et afficher les magasins les plus proches. Elle n’est pas utilisée pour créer un compte, n’est pas utilisée par BOMEDYS NV pour créer des profils marketing individuels et n’est pas enregistrée dans une base de données clients de BOMEDYS NV via ce site web.

Vous pouvez également utiliser le store locator sans partager votre localisation, par exemple en recherchant un code postal, une commune, une ville ou un lieu.

Utilisation de la fonction de recherche

Lorsque vous introduisez un code postal, une ville, une commune ou une adresse, cette recherche peut être transmise à OpenStreetMap/Nominatim afin de convertir le lieu introduit en coordonnées. Ces coordonnées sont utilisées pour afficher les points de vente les plus proches.

Introduisez de préférence un code postal, une commune ou une ville. Il n’est pas nécessaire d’introduire votre adresse privée complète pour utiliser ce site web.

Vous trouverez plus d’informations à ce sujet sur la page OpenStreetMap.

Finalités du traitement

BOMEDYS NV traite des données personnelles uniquement pour les finalités suivantes :

  • afficher le store locator et permettre de trouver des points de vente à proximité ;
  • assurer le bon fonctionnement du site web et du choix de langue ;
  • sécuriser, maintenir et gérer techniquement le site web ;
  • répondre aux questions ou signalements lorsque vous contactez BOMEDYS NV.

Le traitement repose, selon la situation, sur votre consentement, sur l’intérêt légitime de BOMEDYS NV à proposer un site web fonctionnel et sécurisé, ou sur la nécessité de répondre à une demande que vous avez vous-même formulée.

Campagnes Meta

Ce store locator peut être utilisé comme page de destination pour des publicités sur Meta, comme Facebook ou Instagram. BOMEDYS NV ne place sur ce site web aucun Meta Pixel, aucun cookie marketing et aucune technologie de suivi similaire permettant de suivre individuellement les visiteurs sur ce site web.

BOMEDYS NV peut toutefois consulter dans Meta Ads Manager des résultats généraux de campagne, comme le nombre d’impressions publicitaires, de clics ou de clics vers ce site web. Cette mesure se fait au sein de la plateforme publicitaire de Meta et non au moyen d’un code de suivi placé par BOMEDYS NV sur ce site web.

Meta traite les données conformément à sa propre politique de confidentialité lorsque vous utilisez des plateformes Meta telles que Facebook ou Instagram, ou lorsque vous cliquez sur une publicité menant vers ce site web.

Cookies et stockage local

Ce site web n’utilise pas de cookies marketing ni de cookies analytiques de BOMEDYS NV.

La langue choisie peut être enregistrée localement dans votre navigateur via localStorage, afin que le site web s’ouvre de manière cohérente dans la même langue. Il s’agit d’une préférence fonctionnelle.

Vous trouverez plus d’informations dans la politique relative aux cookies.

Services externes

Ce site web peut utiliser des services externes ou renvoyer vers ceux-ci. Ces parties peuvent, dans certains cas, traiter des données techniques, telles que votre adresse IP, lorsque leurs services sont chargés ou lorsque vous cliquez sur un lien externe.

  • GitHub Pages pour l’hébergement du site web ;
  • Google Fonts pour le chargement de la police utilisée ;
  • OpenStreetMap/Nominatim pour convertir les recherches en coordonnées ;
  • Google Maps lorsque vous ouvrez un itinéraire ou une carte ;
  • sites web ou boutiques en ligne de points de vente lorsque vous cliquez vers un site externe.

Lorsque vous ouvrez un site web externe, les conditions de confidentialité et de cookies de cette partie externe s’appliquent. BOMEDYS NV n’a aucun contrôle sur le traitement des données par des sites web externes.

Durée de conservation

BOMEDYS NV ne conserve pas, via ce store locator, de données de localisation ou de recherches dans une base de données clients. La préférence linguistique reste enregistrée dans votre navigateur jusqu’à ce que vous la supprimiez vous-même via les paramètres de votre navigateur ou jusqu’à ce que le navigateur supprime ces données.

Lorsque vous contactez BOMEDYS NV, vos coordonnées et votre message sont conservés aussi longtemps que nécessaire pour traiter votre question ou signalement et, si nécessaire, pour respecter des obligations administratives ou juridiques.

Sécurité

BOMEDYS NV prend des mesures techniques et organisationnelles raisonnables afin de protéger les données personnelles contre la perte, l’abus, l’accès non autorisé ou la modification non autorisée. Le site web est conçu selon le principe de minimisation des données.

Vos droits

Dans les limites de la législation applicable, vous disposez du droit de :

  • demander l’accès à vos données personnelles ;
  • faire corriger des données inexactes ;
  • faire supprimer des données ;
  • demander la limitation du traitement ;
  • vous opposer à certains traitements ;
  • retirer votre consentement lorsque le traitement est basé sur votre consentement ;
  • demander la portabilité des données lorsque cela est légalement applicable.

Vous pouvez exercer ces droits en nous contactant via info@bomedys.com.

Introduire une plainte

Si vous estimez que vos données personnelles ne sont pas traitées correctement, vous pouvez contacter BOMEDYS NV. Vous avez également le droit d’introduire une plainte auprès de l’Autorité de protection des données belge.
