OpenStreetMap et Nominatim

Cette page explique comment ce store locator Joie peut utiliser OpenStreetMap et Nominatim lors de la recherche de points de vente.

OpenStreetMap est une base de données cartographique ouverte. Nominatim est un service de recherche qui utilise les données OpenStreetMap pour convertir un lieu, un code postal, une ville ou une adresse en coordonnées.

Dernière mise à jour : 17 mai 2026.

Quand Nominatim est-il utilisé ?

Le store locator recherche d’abord dans ses propres données de magasins. Lorsque vous saisissez un code postal, une ville, une commune ou une adresse qui ne peut pas être directement associé aux données des magasins, la recherche peut être envoyée à Nominatim.

Nominatim convertit la recherche saisie en coordonnées. Ces coordonnées sont ensuite utilisées pour trier les points de vente par distance et afficher les magasins les plus proches.

Quelles données sont transmises ?

Lorsque Nominatim est utilisé, le terme de recherche saisi peut être transmis. Il peut s’agir, par exemple, d’un code postal, d’une ville, d’une commune ou d’une adresse.

Introduisez de préférence uniquement un code postal, une ville ou une commune. Il n’est pas nécessaire d’introduire votre adresse privée complète pour utiliser ce store locator.

BOMEDYS NV n’utilise pas la recherche saisie pour créer un compte, ne crée pas de profils marketing individuels via ce site web et ne place sur ce site aucun Meta Pixel ni cookie marketing.

Utilisation de votre position actuelle

Si vous cliquez sur « Utiliser ma position actuelle », votre navigateur vous demande d’abord votre autorisation. Sans votre autorisation, ce site web ne peut pas utiliser votre localisation précise.

Votre localisation navigateur est utilisée pour trier les points de vente par distance. Cette localisation n’est pas recherchée via Nominatim et n’est pas enregistrée par BOMEDYS NV dans une base de données clients via ce site web.

Service externe

Nominatim est un service externe. Lorsqu’une recherche est envoyée à Nominatim, l’OpenStreetMap Foundation ou l’exploitant du service peut traiter des données techniques, telles que votre adresse IP, l’heure de la demande et le terme de recherche saisi.

L’utilisation de Nominatim est soumise à la politique de confidentialité et aux conditions d’utilisation de l’OpenStreetMap Foundation. BOMEDYS NV n’a aucun contrôle sur le traitement technique effectué par ce service externe.

Limites d’utilisation

Le service public Nominatim est destiné à une utilisation limitée et raisonnable. Ce store locator est conçu de manière à effectuer des recherches uniquement lorsqu’un visiteur lance lui-même une recherche.

En cas de trafic de campagne élevé ou d’utilisation intensive, il peut être nécessaire d’utiliser une solution de géocodage propre, un système de cache ou un autre service professionnel de recherche de localisation.
